# SW-API

A Java-based API for managing and retrieving Star Wars universe data.

## Description

This project provides a RESTful API to query and manage information about characters, planets, starships, and other elements from the Star Wars universe.

## Features

- Retrieve detailed information about characters and planets.
- REST endpoints built with Java.
- Modular and maintainable code structure.
